MSK Exam Practice (TextEdit version)

This is a static (no-backend) version of your Replit/Flask project.
You can open it locally with any browser.

How to run (Mac):
Double-click index.html (it will open in Safari/Chrome).

Files:
- index.html  (home)
- exam.html   (study page; uses ?exam=shoulder etc.)
- quiz.html   (quiz page; uses ?exam=shoulder etc.)
- msk_data.js (your MSK_EXAMS data, embedded as JSON)
- static/     (images)
